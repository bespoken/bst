normalizeSlots

  • normalizeSlots(utterance: string): void
    • Defined in lib/alexa/sample-utterances.ts:117

    Takes a phrase like "This is a {Slot}" and turns it into "This is a {}" This is so we can compare the sample utterances (which have names that tie off to the slot names defined in the intent schema) with the actual utterance, which have values in the slot positions (as opposed to the names)
